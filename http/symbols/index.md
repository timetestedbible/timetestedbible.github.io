<div class="symbol-intro">
<p>This dictionary catalogs the <strong>symbolic meanings</strong> of biblical words, derived through systematic scriptural analysis. Each symbol has been carefully studied by examining how Scripture itself defines, uses, and applies the term.</p>
<p><strong>Why Symbols?</strong> See <a href="/symbols/how-scripture-teaches/">How Scripture Teaches</a> — the Bible declares it teaches through parables, similitudes, and dark sayings (Hosea 12:10, Matthew 13:34). See also <a href="/symbols/why-parables/">Why Parables?</a> — why symbolic encoding is statistically optimal and how AI confirms the pattern.</p>
<p><strong>Methodology:</strong> See <a href="/symbols/methodology/">How to Study Biblical Symbols</a> for a detailed guide to the analysis process.</p>
<p><strong>The Big Picture:</strong> See <a href="/symbols/architecture/">The Symbolic Architecture of Scripture</a> for how these symbols form a unified, covenant-centered system.</p>
